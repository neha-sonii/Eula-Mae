# Eula Mae's Cafe Website

A modern, story-driven restaurant website designed for **Eula Mae's Cafe**, inspired by family traditions, Southern hospitality, and the legacy of Eula Mae and Penny.

---

## 📖 Overview

Eula Mae's Cafe is more than a restaurant website — it's a digital experience that tells a family's story while helping customers discover the cafe, explore the menu, and make reservations online.

The website combines emotional storytelling, modern design, and practical functionality to create a memorable user experience.

---

## ✨ Features

- Fully responsive design
- Modern cafe landing page
- Online reservation system
- Email notifications
- Story-driven About section
- Mobile optimization
- Fast loading performance
- Contact and location information
- Conversion-focused layout
- Accessibility-friendly design

---

## 🍽️ Reservation System

The reservation system allows customers to submit reservation requests directly through the website.
